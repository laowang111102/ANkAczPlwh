# 音乐管理系统 java817

## 项目概述

音乐管理系统是一个集音乐播放、收藏、下载与用户管理于一体的综合平台。该项目基于 Spring Boot + Vue 技术栈开发，实现了前后端分离，并提供完整的技术文档。

## 技术栈

- 后端：Spring Boot
- 前端：Vue

## 功能模块

### 音乐播放与控制

- 音乐播放
- 歌词同步显示
- 音乐收藏
- 音乐下载
- 拖动控制
- 音量控制

### 歌单与歌曲管理

- 歌曲搜索
- 歌单打分
- 歌单、歌曲评论
- 歌单列表
- 歌手列表分页显示

### 用户管理

- 用户登录注册
- 用户信息编辑
- 头像修改
- 对用户、歌曲、歌手、歌单信息的管理

### 推荐算法

- 用户的协同过滤算法实现

## 系统角色

- 普通用户：使用系统的基本功能
- 管理员：对用户、音乐资源进行管理

## 运行环境

- 后端：Java 1.8 或以上版本，兼容 Spring Boot 的数据库系统（如 MySQL）
- 前端：Node.js，兼容 Vue 的现代浏览器

## 部署步骤

1. 配置后端数据库环境
2. 部署后端 Spring Boot 应用
3. 部署前端 Vue 应用

## 目录结构

```
音乐管理系统
├── 后端（Spring Boot）
├── 前端（Vue）
└── 技术文档
```

- 前后端分离架构，易于维护与扩展
- 提供详尽的技术文档，方便用户与开发者理解和使用系统
